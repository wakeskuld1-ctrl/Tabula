# 截图资源说明

本目录用于存放 README 与文档中引用的界面截图。

## 命名规范

- 首页主图：`frontend-grid-overview.png`
- 关键交互图：`frontend-filter-panel.png`
- 统一使用小写英文与连字符命名

## 建议规格

- 推荐分辨率：`1920x1080` 或 `1600x900`
- 推荐格式：PNG
- 单图建议小于 2MB，保证 GitHub 页面加载速度

## 更新方式

1. 保持文件名不变，直接替换同名图片
2. 提交后检查 `README.md` 图片是否正常显示
3. 若新增图片，请同步更新 README 的引用路径
