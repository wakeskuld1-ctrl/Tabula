# 公式样例 README 设计文档

**日期：2026-03-14**

## 背景与问题
- 用户缺少公式样例参考，难以上手。

## 目标
- 在 README 中补齐已支持公式的样例。
- 中英双语说明，便于理解。
- 按类型分组，便于查找。

## 方案对比
### 方案A（推荐）：README 直接分组展示
**优点**
- 结构清晰、易查找
- 维护成本低

**缺点**
- README 稍长

### 方案B：README 链接到独立文档
**优点**
- README 更简洁

**缺点**
- 用户需要额外点击

## 设计概述（方案A）
### 核心思路
- 在 README 新增“公式样例 / Formula Examples”章节。
- 分为聚合、查找、算术三类。
- 每条样例含公式 + 中文说明 + 英文说明。

### 数据流
1. README 新增分组样例
2. 用户查阅并复制公式

### 测试策略
- 人工检查：样例覆盖已支持函数，文案准确。

## 影响范围
- 仅修改 README

## 回滚策略
- 删除新增章节即可回滚
