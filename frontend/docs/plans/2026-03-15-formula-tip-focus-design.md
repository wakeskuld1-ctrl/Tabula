# Formula Tips Focus Gate Design

## 背景
当前公式提示在“选中含公式单元格”时也会显示，导致交互显得突兀。

## 目标
- 仅在用户开始编辑公式（输入框聚焦且以 `=` 开头）或点击 fx 时显示提示。
- 选中含公式的单元格不触发提示显示。

## 非目标
- 不调整提示内容、样式与折叠交互。
- 不改动 Pivot 逻辑。

## 方案概览
### 方案 A（采用）
- `shouldShowFormulaHelp` 增加 `isFocused` 参数。
- 显示规则：`fx` 打开时总显示；否则仅在 `输入框聚焦 && 文本以 '=' 开头` 时显示。

### 方案 B
- 仅当“用户输入导致内容变更”时显示，忽略选中带来的值变更。

### 方案 C
- 上层传入“值来源（selection/user-edit）”，只对 user-edit 显示。

## 设计细节
- `FormulaBar` 增加 `isFocused` 状态与 `onFocus/onBlur` 事件。
- `shouldShowFormulaHelp` 使用 `isFocused` 控制提示出现。
- 新增测试：
  - `=` 但未聚焦 -> 不显示
  - `=` 且聚焦 -> 显示
  - fx 打开（未聚焦） -> 显示

## 验收标准
- 选中含公式单元格不会展示提示。
- 在输入框聚焦并输入 `=` 后提示出现。
- 点击 fx 按钮时提示出现（无需聚焦）。
