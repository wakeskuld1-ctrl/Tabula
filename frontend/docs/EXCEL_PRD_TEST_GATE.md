# Excel 类表格能力建设测试门禁清单

- 来源需求: [Excel 类表格能力建设产品需求文档 (PRD).md](file:///d:/Rust/metadata/Documentation/00_Project_Management/Requirements/Excel%20类表格能力建设产品需求文档%20(PRD).md)
- 文档用途: 合并前测试门禁数据
- 验收规则: 仅当 P0 全绿、P1 无阻断红灯、风险项已评估且可接受时，允许合并
- 红绿灯说明:
  - 绿灯: 通过
  - 红灯: 失败
  - 黄灯: 有风险，需记录并评估是否阻断

## 1. 门禁通过标准

| 维度 | 门禁标准 | 是否阻断合并 |
| --- | --- | --- |
| P0 功能 | 全部用例为绿灯 | 是 |
| P1 功能 | 可有黄灯，不可有阻断级红灯 | 是 |
| 风险项 | 循环依赖、超时熔断、异常恢复有明确结论 | 是 |
| 业务流程 | 上传 Excel / 新建沙盘 / 保存沙盘 / 时光轴 关键流程通过 | 是 |
| 回归结果 | 核心链路回归通过 | 是 |
| 缺陷状态 | 阻断级缺陷全部关闭 | 是 |

## 2. 测试数据基线

- Sheet1: A1=10, A2=20, A3 空, B1="abc"
- Sheet2: A1=5, A2=15
- 特殊 Sheet 名: `Sheet 3`, `统计-2026`
- 公式样例:
  - 正常: `=SUM(A1:A2)`, `=Sheet2!A1+A1`, `=IF(A1>0,1,0)`
  - 异常: `=SUM(`, `=NotExistFn(A1)`, `=SheetX!A1`

## 6. 缺陷分级建议

- S1 阻断: 核心链路不可用（公式同步失败、跨表计算错误、崩溃）
- S2 严重: 功能可用但结果不稳定（Undo/Redo 错乱、持久化丢失）
- S3 一般: 体验问题（提示交互抖动、轻微延迟）
- S4 建议: 优化项（提示文案、边界输入易用性）
