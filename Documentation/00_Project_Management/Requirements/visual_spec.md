# 视觉设计规范（深色科技感 Excel Demo）

## 色彩板

- 主色（霓虹蓝）：#5CE1FF
- 辅助色（霓虹紫）：#8B5CF6
- 背景色（深空黑）：#0A0F1C
- 面板底色（深蓝灰）：#0B1222
- 描边色（冷灰边界）：rgba(148,163,184,0.20)
- 文本主色：#E5E7EB
- 文本次色：#94A3B8
- 状态色：成功 #22D3EE，错误 #F87171

## 排版

- 字体家族：-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif
- 标题字重：700，字号 16px
- 辅助标签：12px，字重 600
- 正文输入：14px
- 信息提示：10–12px

## 间距与布局

- 顶部状态栏高度：48px，左右内边距 16px
- 工具栏高度：44px，控件间距 6–12px
- 公式栏高度：34px
- 左侧栏宽度：260px（展开），68px（收起）
- 左侧栏分区间距：12px，区块圆角 12px
- 主工作区：外间距 12px，圆角 16px
- 页面主容器：全高布局，背景使用双重径向渐变叠加

## 交互与质感

- 面板使用半透明深色玻璃质感与模糊背景
- 按钮悬停出现轻微外发光与上浮
- 重要状态使用霓虹强调色与柔和光晕
